Java的值传递机制

发表于 2020-10-27 分类于 Java

Java中方法的值传递方式只有一种：值传递。

值传递机制：

  • 如果参数是基本数据类型，此时实参赋给形参的是实参真实存储的数据值。

实参a,b将值传给形参，排序后a,b的值未发生改变，对实参值没有影响。

内存中

  • 如果参数是引用数据类型，此时实参赋给形参的是实参存储数据的地址值。

内存中显示

参数保存的是实际对象的地址值，改变的只是地址值指向的堆内存中的实际对象，并没有真正改变参数，参数的地址值没有变。
